 
# Intranät för kyrkor
Till att börja med kommer jag bygga ett schemläggningssystem för ungdomsverksamhet och likande där man kan lägga in vilka personer som är på plats och vilka som är ansvariga osv.
Tanken är att man ska kunna få ut både gemensamma och enskilda kalendrar som ics för google kalender och liknande.

Systemet är byggt på fastapi och vue quasar.
